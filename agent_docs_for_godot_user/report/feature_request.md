Copy this block into the developer Discord first:

````text
Title:

Requested feature:

Current workaround:

Desired Godot API shape or usage example:
```gdscript

```

Related module:

Priority:
- blocker / high / medium / low

Compatibility impact:
- none known / may affect existing Godot API / may affect saved data / unknown

Deployment or packaging impact:
- none known / requires new runtime files / affects exports / unknown

Documentation impact:
- no docs needed / update existing docs / add new example / unknown

Private assets needed to explain the request?
- no / yes, can share privately / yes, cannot share

Other notes:
````

For API requests, include the smallest Godot code example that would make the
feature useful. For documentation requests, include the current confusing page
or missing workflow.

Do not post private music, database, or model files publicly unless you have
the right to share them. Use a minimal substitute whenever possible. A human
developer must directly review and send the final Discord or GitHub message.
Developers welcome feature requests, and agent-generated request drafts are
fine when they make reporting easier. Automated reporting or unreviewed message
sending is discouraged because the request may expose personal information,
local paths, private assets, database content, logs, or model files.
